# InfiNote MVP 驗收標準（iOS Apple Pencil 筆記 App）

## 1) 功能驗收
- Apple Pencil 書寫：僅 Apple Pencil 可觸發筆劃，手指只用於平移/縮放，不產生墨跡。
- 筆工具：至少提供鉛筆、原子筆、螢光筆三種筆刷，且可調整顏色、粗細、透明度。
- 橡皮擦：支援 `Stroke Eraser`（整筆刪除）與 `Segment Eraser`（局部切段）。
- 套索工具：可圈選多筆劃與文字方塊，並可執行移動、縮放、刪除、合併。
- 無邊際畫布：可連續平移與縮放，不出現畫布邊界或超大底圖載入等待。
- 文字方塊：可插入、拖拉、修改字型/字級/顏色，並與手寫內容共存。
- PDF 匯入：可從檔案 App 多選匯入 PDF；每份 PDF 生成一個筆記本。
- PDF 標註：PDF 為底層、手寫與文字為上層，切頁後標註位置正確。
- 匯出能力：支援 PDF（紙張大小、邊距、密碼）、PNG/JPG、RTF（僅文字）。
- 文件格式：可讀寫 `.note` 自訂格式，包含筆劃、文字、PDF、圖片、字型資產。
- Home 管理：支援資料夾/筆記本建立、PDF 批次匯入、縮圖與最後編輯時間顯示。

## 2) 效能驗收
- 書寫延遲：一般書寫情境下，筆尖跟隨感需穩定且無可感知拖影；主觀評估達 GoodNotes 級別。
- 幀率目標：書寫與拖曳縮放時，畫面更新穩定，常態維持 60 FPS（A-series iPad 近代機型）。
- 大量筆劃：單頁 10,000+ 筆劃仍可平移/縮放/選取，無明顯卡死或長時間無回應。
- 記憶體控制：連續操作 30 分鐘不發生記憶體暴增與系統回收重啟；縮圖快取可受控釋放。
- 存檔速度：背景自動存檔不阻塞書寫；高頻編輯下不出現可見 UI 卡頓。
- 重開速度：App 重啟後可快速恢復最近頁面內容，使用者可在短時間內繼續書寫。

## 3) 使用情境驗收
- 情境 A（課堂筆記）：
- 使用者連續書寫 15 分鐘、切換筆刷與橡皮擦、插入文字方塊，過程無中斷與明顯延遲。
- 情境 B（PDF 批註）：
- 一次匯入 3~10 份 PDF，逐頁標註後匯出新 PDF，內容與版面對齊正確。
- 情境 C（整理編輯）：
- 使用套索選取多筆劃移動/縮放/合併，結果可預期且可持續操作不崩潰。
- 情境 D（文件交換）：
- 將筆記匯出為 `.note` 後再匯入，筆劃、文字、資產、頁面結構保持一致。
- 情境 E（分享交付）：
- 匯出 PDF/PNG/JPG/RTF 後可直接呼叫系統分享與 AirPrint，流程可一次完成。
- 情境 F（中斷恢復）：
- App 被系統切到背景或強制關閉後，重新開啟可恢復最近編輯狀態，資料不遺失。

## 4) 驗收結論判定（建議）
- 必要條件：功能驗收項目全數通過。
- 效能條件：關鍵指標（書寫流暢、幀率穩定、重開快速）全部達標。
- 場景條件：6 個使用情境至少 5 個一次通過，且不得有資料遺失或崩潰。
